# 🔍 ИНСТРУКЦИЯ: Добавление сайта в поисковые системы

## ЧАСТЬ 1: Google Search Console

### Шаг 1: Регистрация
1. Перейти на https://search.google.com/search-console
2. Нажать кнопку **"Начать"** или **"+ Добавить свойство"**
3. Выбрать **Свойство URL**
4. Ввести: `https://evrocontayner.kz`
5. Нажать **Продолжить**

### Шаг 2: Подтверждение владения сайтом
Выбрать один из способов (рекомендуется **DNS**):

#### Способ A: Подтверждение через DNS (рекомендуется)
1. В Google Search Console выбрать **Подтверждение через DNS**
2. Скопировать код (выглядит примерно так: `google-site-verification=XXXXXXXXXXXXX`)
3. Перейти в панель управления доменом (где зарегистрирован evrocontayner.kz)
4. Найти раздел **DNS записи** или **DNS Management**
5. Добавить новую TXT запись:
   - **Тип**: TXT
   - **Имя/Host**: @ или оставить пустым
   - **Значение**: Вставить скопированный код
6. Сохранить изменения
7. Вернуться в Google Search Console
8. Нажать **Подтвердить**
9. Дождаться проверки (обычно 15-60 минут)

#### Способ B: Подтверждение через HTML файл (если DNS не работает)
1. В Google Search Console выбрать **Подтверждение через HTML файл**
2. Скачать файл `google[код].html`
3. Загрузить его в корневую папку сайта (/public_html или /)
4. Проверить доступность: https://evrocontayner.kz/google[код].html
5. Вернуться в Google Search Console
6. Нажать **Подтвердить**

### Шаг 3: Добавление Sitemap
1. После подтверждения владения, перейти в левом меню в **"Карты сайта"** (Sitemaps)
2. Нажать **"+ ДОБАВИТЬ КАРТУ САЙТА"**
3. Ввести: `sitemap.xml`
4. Нажать **Отправить**
5. Дождаться обработки (статус "Успех" должен появиться через несколько минут)

### Шаг 4: Проверка индексации
1. В левом меню нажать **"Охват"** (Coverage)
2. Проверить статусы страниц:
   - 🟢 **Valid** - страница успешно проиндексирована
   - 🟠 **Valid with warnings** - проиндексирована, но есть ошибки
   - 🔴 **Error** - не проиндексирована

### Шаг 5: Мониторинг и оптимизация
Регулярно проверять в Google Search Console:

1. **"Результаты поиска"** (Performance)
   - Количество кликов из поиска
   - среднюю позицию
   - Рейтинг кликов (CTR)
   - Какие запросы приводят трафик

2. **"Улучшения"** (Enhancements)
   - Проверить структурированные данные
   - Исправить ошибки расширенных результатов

3. **"Экспериментальные"** (Candidate Features)
   - Core Web Vitals
   - Mobile usability
   - Безопасность

### Шаг 6: Отправка на переиндексацию
Для срочной индексации новых или обновленных страниц:
1. В **"Охват"** → **"Запросить индексацию"**
2. Ввести URL страницы
3. Нажать **"Проверить и протестировать"**
4. После проверки нажать **"Запросить индексацию"**

## ЧАСТЬ 2: Яндекс.Вебмастер

### Шаг 1: Регистрация
1. Перейти на https://webmaster.yandex.ru
2. Кнопка **"Добавить сайт"** (вверху справа)
3. Ввести: `https://evrocontayner.kz`
4. Нажать **"Добавить сайт"**

### Шаг 2: Подтверждение владения сайтом
Выбрать один из способов (рекомендуется **Яндекс.Метрика**):

#### Способ A: Подтверждение через Яндекс.Метрику (самый простой)
1. Убедиться что на сайте установлен код Яндекс.Метрики
2. В Яндекс.Вебмастер выбрать **"Подтверждение через Яндекс.Метрику"**
3. Нажать **Подтвердить**
4. Проверка произойдет автоматически (обычно 15-60 минут)

#### Способ B: Подтверждение через Meta-тег
1. Выбрать **"Подтверждение через Meta-тег"**
2. Скопировать тег (выглядит: `<meta name="yandex-verification" content="XXXXX" />`)
3. Добавить в `<head>` всех HTML файлов или хотя бы главной страницы
4. Сохранить файл
5. Вернуться в Яндекс.Вебмастер
6. Нажать **Подтвердить**

#### Способ C: Подтверждение через DNS
1. Выбрать **"Подтверждение через DNS"**
2. Скопировать значение
3. Добавить TXT запись в DNS (как в Google Search Console)
4. Подтвердить в интерфейсе

### Шаг 3: Загрузка Sitemap
1. Перейти в **"Мои сайты"** → выбрать сайт
2. В левом меню **"Индексирование"** → **"Карты сайта"** (Sitemap)
3. Нажать **"Добавить карту сайта"**
4. Ввести: `sitemap.xml`
5. Нажать **Добавить**

### Шаг 4: Проверка индексации
1. В **"Индексирование"** → **"Статус индексации"**
2. Проверить количество проиндексированных страниц
3. Убедиться что нет ошибок при индексировании

### Шаг 5: Проверка микроданных
1. Перейти в **"Оптимизация"** → **"Метаданные"**
2. Нажать **"Проверить микроданные"**
3. Если есть ошибки, исправить структурированные данные в HTML

### Шаг 6: Настройка дополнительных параметров

#### Информация о сайте
1. **"Информация о сайте"** → **"Редактировать информацию"**
2. Заполнить:
   - Название сайта: "Evrocontayner"
   - Описание
   - Телефон
   - Адрес
   - Часы работы

#### Ссылки между регионами
1. **"Информация о сайте"** → **"Города"**
2. Если у компании филиалы в разных городах, добавить:
   - `/city/almaty/` → Алматы
   - `/city/astana/` → Астана
   - Etc.

### Шаг 7: Запрос переиндексации
1. **"Индексирование"** → **"Переиндексировать сайт"**
2. Или для отдельной страницы:
3. **"Индексирование"** → **"Запросить переиндексацию"**
4. Ввести URL страницы
5. Нажать **"Переиндексировать"**

- **12 месяцев**: Стабилизация и доминирование по локальным запросам
